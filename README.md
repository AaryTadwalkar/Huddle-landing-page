### What I learned

This project helped me improve multiple core frontend skills:

- Flexbox layout handling
  - Learned how to properly align sections side-by-side and center content
- Responsive design
  - Used media queries (max-width: 375px) to adapt layout for mobile
- Background handling
  - Switching between desktop and mobile SVG backgrounds
- Button hover effects
  - Smooth transitions using transition: all 0.3s ease
